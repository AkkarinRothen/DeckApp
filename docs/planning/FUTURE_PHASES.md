# Plan: Fases Futuras y Experimentos (Fase 4+)

Ideas y bosquejos para la evolución a largo plazo de DeckApp.

## Especialización de Layouts (B-5)
- **Spread Tarokka:** Layout de 5 posiciones fijas con significados específicos.
- **Relojes de Progreso (PbtA):** Integración de relojes visuales en la sesión.

## Inteligencia Visual (B-7)
- **OpenCV Auto-grid:** Usar visión artificial para detectar cartas en una página de PDF automáticamente.
- **Detección de Bordes:** Mejorar el recorte automático de escaneos de manuales reales.

## Cloud & Social (E-1, E-6)
- **Supabase Sync:** Sincronización opcional de la biblioteca entre dispositivos.
- **Modo Espectador:** URL para que los jugadores vean las cartas reveladas en tiempo real.

## Branding
- **Renombramiento:** Evaluación de nombres (Arcana, Grimoire, Vault).
- **Iconografía personalizada:** Pack de iconos TTRPG consistente.
